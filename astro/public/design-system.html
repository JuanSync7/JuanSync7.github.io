CHART TYPES — CLICK CODE TO COPY
bar chart
Live animated bars, gradient fill bottom→top. Used for skill growth + AI throughput.
const [c1, c2] = GRAD.roseGold;
const grad = ctx.createLinearGradient(0, h, 0, 0);
grad.addColorStop(0, c1);
grad.addColorStop(1, c2);
ctx.fillStyle = grad;
ctx.fillRect(x, y, barW, barH);
speedometer
Half-circle gauge, animated needle, tick marks. Needle: val += (target - val) * 0.04
const [c1, c2] = GRAD.greenYellow;
ctx.arc(cx, cy, r, Math.PI, 2 * Math.PI);
const grad = ctx.createLinearGradient(cx-r, cy, cx+r, cy);
grad.addColorStop(0, c1);
grad.addColorStop(1, c2);
ctx.strokeStyle = grad; ctx.lineWidth = 18;
ctx.lineCap = 'round'; ctx.stroke();
donut chart
Ring chart with gradient segments, gap between slices, leader lines. Inner/outer ratio: 0.58
const [c1, c2] = GRAD.emeraldOrange;
ctx.arc(cx, cy, outerR, angle, angle + sweep);
ctx.arc(cx, cy, innerR, angle + sweep, angle, true);
ctx.closePath();
const grad = ctx.createLinearGradient(gx1,gy1,gx2,gy2);
grad.addColorStop(0, c1); grad.addColorStop(1, c2);
ctx.fillStyle = grad; ctx.fill();
circular gauge
Full-circle progress ring with diagonal linear gradient.
const [c1, c2] = GRAD.cyanMagenta;
ctx.arc(cx, cy, r, -PI/2, -PI/2 + frac * PI * 2);
const grad = ctx.createLinearGradient(cx-r,cy-r,cx+r,cy+r);
grad.addColorStop(0, c1);
grad.addColorStop(1, c2);
ctx.strokeStyle = grad; ctx.lineWidth = 6;
ctx.lineCap = 'round'; ctx.stroke();

08 — bento grid layout
Overview page uses an asymmetric 2-column bento grid. Each section lives in a tile with subtle bg + border. Full-width tiles span both columns.
GRID STRUCTURE
.cp-bento — display: grid; grid-template-columns: 2fr 3fr; gap: 14px
.cp-tile — base tile: rgba(255,255,255,0.02) bg, 1px border, 10px radius, 14px 16px padding
.cp-tile-full — spans both columns (grid-column: 1 / -1)
.cp-tile-a / -b — left (narrow) / right (wide) column (row 2: output rate + proficiency)
.cp-bento-row3 — sub-grid row: grid-template-columns: 3fr 2fr (skill growth wide, commit streak narrow)
LAYOUT MAP
┌──────────────────────────────────┐ │ vitals (stat cards) ▸ full │ ├──────────┬───────────────────────┤ │ output │ proficiency │ │ rate │ (donut chart, large) │ │ (small) │ │ ├──────────┴──────┬────────────────┤ │ skill growth │ commit streak │ │ (bar chart) │ (heatmap) │ ├─────────────────┴────────────────┤ │ ticker + explore ▸ full │ └──────────────────────────────────┘
DONUT TEXT SIZING
Category labels: bold 15px · Skill counts: bold 13px #c0d8cc · Center total: bold 32px · Sub-label: bold 13px
RESPONSIVE
≤640px: single column stack, all tiles and sub-grids collapse to full-width
